Metodología

Esta página web se ha creado transformando los archivos de mis prácticas en lenguaje Markdown a lenguaje HTML mediante la herramienta Pandoc. Se han creado nuevos archivos HTML mediante nano en los que se han trasladado los contenidos de los trabajos y éstos se han aglutinado en una carpeta denominada docs en mi repositorio. Junto a esto, mediante la biblioteca de ejemplos Bootstrap se ha puesto a punto la web con las plantillas que han permitido darle a mi página la apariencia deseada.

La carpeta docs cuenta así con estos componentes:

  • index.html: es el archivo plantilla de la web, en el que figura el índice de contenidos de la web.
  • css: es una carpeta que contiene dos archivos css (bootstrap.min.css y stick-footer-navbar.css) que han servido para la configuración de la web, principalmente de su estilo.
  • js: es otra carpeta que contiene un archivo javascript (bootstrap.bundle.min.js)
  • cabecera.html
  • practica1.html
  • practica2.html
  • practica3.html
  • pie.html
  • img: una carpeta con las imágenes que aparecen insertadas en mis trabajos.

    Para que todo esto se pudiese visualizar como una web, desde Settings en Github se ha accedido a Github Pages y se ha seleccionado como rama principal docs para así poder visualizar mi web.

    Una vez hecho esto, se ha llevado a cabo un trabajo de refinamiento y pulido de la página. Se han editado las plantillas de Bootstrap para crear una barra superior en la que apareciesen cada una de las prácticas como una entrada diferente usando esta fórmula: li class=“nav-item”> a class=“nav-link” href=“practica2.html”>Practica 2. Además, se ha editado la plantilla para que se pudiese retornar al inicio. También se ha editado el pie de la página para que mostrase UC3M.

    Por último, se ha revisado que las rutas de las imágenes fuesen correctas y se han realizado las correcciones pertinentes en los archivos ante posibles fallos en la transformación del lenguaje md a HTML.

    Una vez hecho todo esto se ha trasladado del repositorio local el de GitHub y…¡ya está lista mi página!
